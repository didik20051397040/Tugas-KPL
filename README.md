


## Cara install

- `git clone https://github.com/didik20051397040/Tugas-KPL.git`
- `cd Tugas-KPL`
- `composer install`
- `cp .env.example .env`
- Ubah file env.example -> env dan buat database kpltugas
- `php artisan key:generate`
- `php artisan migrate`
- `php artisan db:seed`
- `php artisan passport:install`
- `npm install`
- `npm run dev`
- `php artisan serve`
